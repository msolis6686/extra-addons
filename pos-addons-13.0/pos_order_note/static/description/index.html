POS Advanced Order Notes

Add notes to the entire order, use already predefined notes and speed up the process of creating orders by specifying products also via notes, which can be automatically applied further.

Key features:
  • Set predefined notes for separate products as well as for entire order
  • Do not waste your time for specifying products/order notes manually
  • Create any number of notes as much as you like
  • Apply several notes simultaneously
  • Add notes to the order automatically

How It Works

Go to Point of Sale → Configuration → POS Product Notes menu and create your notes.

In POS notes are displayed according to the sequence value.

Product Notes

Open POS session and click on the "Note" button. By default you will see Product Notes

Order Notes

To add a note to entire order switch to Order Note

Click on the "Other" button to see all predefined notes

Order printer receipt

Click on the "Order" button and see the product/order notes on receipt printed on the Order printer

Auto-applied notes for the order

Open product form and go to the "Notes" tab. Here you can set note that will be applied automatically, once you add this product to the order in POS.
