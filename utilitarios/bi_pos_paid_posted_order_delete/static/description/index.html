POS - Point of Sales Delete Order Odoo Apps

Delete Paid & Posted Orders on POS Odoo Apps

Point of Sales Order Cancel and Delete Odoo Apps

POS - Point of Sale Delete Order Odoo App is used to delete paid/posted point of sale orders for that user can configure radio options like 1)Delete POS Order Without Code and 2) Delete POS Order With Code. Users can delete single or multiple paid/posted pos order using with code or without code.

Features

Delete POS Order Without Code

User can delete single or multiple point of sale orders without using code.

Delete POS Order With Code

User can delete single or multiple point of sale orders with using enter code.
Demo Request Video Tutorial

User Configuration for POS Delete Order Access

Under user configuration user can tick on "POS Delete Order Access" group.

POS Configuration for Delete Paid Order

Goto Point of Sale -> Configuration -> Settings user can tick on "Delete Paid Order" field in that radio option like 1)Delete POS Order Without Code and 2)Delete POS Order with Code.

Here user can tick on "Delete POS Order Without Code" option.

Delete Paid Single POS Order

User can delete paid single pos order in form view.

Delete Paid Multiple POS Order

User can delete paid multiple pos orders in tree view.

Delete Paid Order Wizard

After select pos orders user will get wizard where they can confirm process of delete an orders.

After Delete Paid POS Orders View

POS Configuration for Delete Paid Order

Goto Point of Sale -> Configuration -> Settings user can tick on "Delete Paid Order" field in that radio option like 1)Delete POS Order Without Code and 2)Delete POS Order with Code.

Here user can tick on "Delete POS Order With Code" option and enter code.

Delete Paid Single POS Order

User can delete paid single pos order in form view.

Delete Paid Multiple POS Order

User can delete paid multiple pos orders in tree view.

Delete Paid Order Wizard

After select pos orders user will get wizard where they can enter code for process of delete an orders.

After Delete Paid POS Orders View

Warning/Validation for Wrong Security Code

When user entered wrong security code then user will get validation message, Enter code is not valid.
